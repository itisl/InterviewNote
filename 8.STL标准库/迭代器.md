
| 容器               | 迭代器功能   |
| :----------------- | :----------- |
| `vector`           | 随机访问     |
| `deque`            | 随机访问     |
| `list`             | 双向         |
| `set` / `multiset` | 双向         |
| `map` / `multimap` | 双向         |
| `stack`            | 不支持迭代器 |
| `queue`            | 不支持迭代器 |
| `priority_queue`   | 不支持迭代器 |

`advance(p, n)`：使迭代器 p 向前或向后移动 n 个元素。
`distance(p, q)`：计算两个迭代器之间的距离，即迭代器 p 经过多少次 + + 操作后和迭代器 q 相等。如果调用时 p 已经指向 q 的后面，则这个函数会陷入死循环。
`iter_swap(p, q)`：用于交换两个迭代器 p、q 指向的值。
