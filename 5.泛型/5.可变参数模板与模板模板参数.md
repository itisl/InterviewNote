## 可变参数模板
```cpp
// 出口函数
void print(){}

template<typename T, typename... Types>
void print(const T& firstArg,const Types&... Args){
    std::cout << firstArg << endl;
    print(Args...);
}
```

> - `...`用于模板参数，是一个模板参数包
> - `...`用于函数参数，是一个函数参数包
> - `...`用于函数参数类型，是一个函数参数类型包
> - `sizeof...(Args)`，参数数量
