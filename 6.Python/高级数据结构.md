| 数据结构/算法 | 语言内置                          | 内置库                                            |
| ------------- | --------------------------------- | ------------------------------------------------- |
| 线性结构      | `list`(列表)/`tuple`(元祖)        | `array`(数组,不常用)/`collection.namedtuple`      |
| 链式结构      |                                   | `collections.deque`(双端队列)                     |
| 字典结构      | `dict`(字典)                      | collections.Counter(计数器)/OrderedDict(有序字典) |
| 集合结构      | `set`(集合)/frozenset(不可变集合) |
| 排序算法      | `sort()`                          |
| 二分算法      |                                   | `bisect`模块                                      |
| 堆算法        |                                   | `heapq`模块                                       |
