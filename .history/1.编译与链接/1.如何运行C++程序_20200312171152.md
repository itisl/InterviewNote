### 使用`g++`命令行编译链接
#### 1. 编译
> `-S`命令将文件转换为汇编文件，生成Test.s文件
```sh
g++ -S Test.cpp
```
#### 2. 目标代码
> `-c`命令将文件转换为目标文件，生成Test.o文件

```sh
g++ -c Test.cpp
```
#### 3. 生成库文件
##### 动态库
##### 静态库

#### 4. 链接
> `-o`命令生成可执行文件Test(或Test.exe)，`-L`命令链接库文件

```sh
g++ Test.cpp -o name -L /usr/include/iostream
```
