5.2 模板特化与偏特化

模板特化

作用：也称为模板全特化，可以对某些特定的类型进行特殊的处理(例如某个算法对int类型的点有更快的处理方法)

template<class Key>
struct hash{};
// 对int类型进行模板特化
template<>
struct hash<int>{
    // ...
};

模板偏特化

参数偏特化

作用：多个模板参数时，对其中某个模板参数进行特化

template<class TypeA, class TypeB=int>
class Point{
    // ...
};
// 对TypeA进行偏特化
template<class TypeB=int>
class Point<bool, TypeB>{
    // ...
};

范围偏特化

作用：可以对模板参数的某种类型(例如指针)进行特化

template<class T>
class C{};
// 对指针类型进行偏特化
template<class T>
class C<T*>{
    // ...
};
// 两个对象初始化将执行不同地方的代码
C<int> obj1;
C<int*> obj2;
