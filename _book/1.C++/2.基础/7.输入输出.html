2.7 输入输出

cin与cout:

getline(istream is,string& str,结束符="\n")

// 已知个数数据输入，以空格隔开, \n 结束
while(cin>>a>>b){
  cout << a + b ;
}
// 多行数据，每行开头指定输入数据个数
while(true){
  int n = 0, e = 0;
  cin >> n;
  while(n--){
    cin >> e;
  }
}

// 多行数据，每行不定个数

while(getline(cin, s, ",")){
  stringstream ss{s};
  while(ss >> e){
    stoi(e); //如果
  }
}
  1. endl等价于'\n'<<flush，会刷新缓冲区，造成效率低下。推荐用'\n'代替endl；

  2. 可以使用tie()函数解除cin与cout的绑定，加快输入输出效率：

      std::ios::syn_with_stdio(false); //解除stdio的兼容
  std::cin.tie(nullptr);    
  std::cout.tie(nullptr);

  3. 输出精度控制： 

    cout.precesion(3);//3为有效数字
